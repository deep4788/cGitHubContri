View GitHub Commits
===================

![](images/appImage.png)

A Google Chrome extension that displays monthly GitHub commits in your contributions graph.

Install
-------

Getting Started
---------------
Within Google Chrome go to:

- chrome://extensions
- Enable Developer Mode

- Test
